# CSS em Escala

> Objetivo: dominar CSS em nível profissional, com foco em component libraries, design systems e interfaces de alta precisão.

---

## 1. Fundamentos obrigatórios

Domine:

- cascade;
- specificity;
- inheritance;
- box model;
- display;
- position;
- stacking context;
- z-index;
- flexbox;
- grid;
- responsive design;
- media queries;
- container queries;
- CSS variables;
- animations;
- transitions;
- `prefers-reduced-motion`;
- dark mode;
- typography.

---

## 2. CSS Architecture

Estude e pratique:

- BEM;
- CSS Modules;
- Styled Components;
- utility classes;
- design tokens;
- cascade layers;
- component-scoped styles;
- global reset;
- theme system.

---

## 3. Projeto 1 — Layout XXX-style

Crie uma página de produto com:

- hero section;
- typography forte;
- grid responsivo;
- imagens grandes;
- animações sutis;
- espaçamento preciso;
- dark/light mode.

---

## 4. Projeto 2 — Sistema de tokens

Crie tokens para:

- cores;
- spacing;
- radius;
- shadow;
- typography;
- breakpoints;
- motion.

Exemplo:

```css
:root {
  --space-1: 4px;
  --space-2: 8px;
  --radius-md: 12px;
  --font-size-md: 1rem;
}
```

---

## 5. Projeto 3 — Componentes com CSS Modules

Crie:

- Button;
- Card;
- Input;
- Modal;
- Tabs;
- ProductCard.

Cada componente deve ter:

- estados;
- variantes;
- responsividade;
- foco visível;
- suporte a tema.

---

## 6. Projeto 4 — Refatoração de CSS ruim

Pegue uma tela com CSS bagunçado e refatore para:

- nomenclatura clara;
- tokens;
- componentes;
- menos especificidade;
- menos repetição;
- comportamento responsivo previsível.

---

## 7. Performance CSS

Estude:

- animações com `transform` e `opacity`;
- evitar layout thrashing;
- reduzir CSS não usado;
- critical CSS;
- fontes otimizadas;
- imagens responsivas.
