# Testes e Qualidade Front-End

> Objetivo: provar que seus componentes não apenas funcionam visualmente, mas são confiáveis, acessíveis e sustentáveis.

---

## 1. Ferramentas

Estude:

- Vitest ou Jest;
- React Testing Library;
- Playwright;
- Cypress;
- axe;
- eslint;
- prettier;
- type checking;
- GitHub Actions.

---

## 2. Tipos de teste

### Testes unitários

Para:

- validação;
- formatação;
- cálculos;
- transformação de dados.

### Testes de componente

Para:

- renderização;
- estados;
- eventos;
- props;
- acessibilidade básica.

### Testes E2E

Para:

- login;
- busca;
- formulário;
- modal;
- navegação;
- carrinho.

### Testes de acessibilidade

Para:

- labels;
- roles;
- contraste;
- teclado;
- ARIA.
